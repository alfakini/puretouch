== Puretouch

Puretouch é uma re-implementação da interface do Puredata utilizando o framework para interfaces multi-toque pymt, e o módulo topd, que implementa funções e objetos do Puredata em Python.

== Install

$ git clone http://github.com/tito/pymt.git

$ cd pymt/

$ sudo python setup.py install

$ cd ../

$ git clone http://github.com/alfakini/puretouch.git

$ cd puretouch/

$ git clone http://github.com/automata/topd.git
